## 1. プロジェクト概要

### 1.1 サイト名

**Add Philia（アドフィリア）**

### 1.2 理念（Philosophy）

Add Philiaは、「Add（加える）」と「Philia（隣人愛・友愛）」を核にした、個人事業としての表現活動です。
効率と論理が優先されがちな時代に、"情愛"という人間の温度を、日常へ静かに編み戻していきます。

### 1.3 Mission（社会的意義）

**無機質な世界に、体温を1℃加える。**

便利さが増すほど、心が置き去りになる瞬間がある。
Add Philiaは、その精神的な空白にそっと寄り添い、明日を生きるための**生存リソース（活力）**を届けることを目的とします。

それは単なる娯楽ではなく、不安や孤独による消耗をやわらげ、もう一度"プラス"へ向かうための支えとなること。
声と言葉と映像を通じて、個人の内側に灯る回復力を守る「見えないセーフティネット」をつくります。

### 1.4 目的

代表NEKOmaloのポートフォリオサイトとして、Add Philiaの理念・活動内容・制作実績を紹介し、各関連プロジェクトへの導線を提供する。訪問者がAdd Philiaの世界観を体感し、問い合わせや各活動への理解を深められるサイトを構築する。

### 1.5 ターゲットユーザー

- Add Philiaの理念に共感する個人・企業
- 動画・アート制作の依頼を検討している方
- VTuber・メタバース関連の仕事依頼を検討している企業・個人
- 東方Project二次創作に興味のあるファン
- YUKImaloのファン・応援者
- 海外からのアクセス者（英語対応）

### 1.6 デザインコンセプト

**「猫 × ボタニカル × ノスタルジック」**

BOTANISTサイトを参考に、シンプルで洗練されたデザインを基調としつつ、猫のモチーフと植物のイラストを融合。柔らかな温もりと懐かしさを感じるビジュアルで、Add Philiaの「情愛」というテーマを視覚的に表現する。訪問者が心地よく、ゆっくりと各ページを巡れる体験を提供する。

## 7. 素材制作・調達ガイドライン

### 7.1 素材制作方針

サイトで使用する画像・動画素材は、以下の方法で制作・調達する。

| 方法 | 用途 | 備考 |
|------|------|------|
| Nano Banana（AI画像生成） | メインビジュアル、背景画像、装飾イラスト、猫イラスト | 商用利用可。高品質な画像を迅速に生成可能 |
| ボタニカルイラスト フリー素材 | フレーム、仕切り線、装飾パーツ | 商用利用可のサイトから調達 |
| 自作・既存素材 | プロフィール画像、作品サムネイル | クライアント提供 |

### 7.2 Nano Banana 活用ガイド

| 項目 | 内容 |
|------|------|
| サービス | Nano Banana Pro（Google Gemini 画像生成機能） |
| 公式URL | https://gemini.google/overview/image-generation/ |
| 利用方法 | Google AI Studio または Gemini API経由 |
| 料金目安 | 無料枠あり（1日100枚まで）。API利用は1K/2K画像で約0.134ドル/枚 |
| 商用利用 | 可能（Gemini API規約により画像の権利はユーザーに帰属） |

### 7.3 生成する素材と推奨プロンプト例

| 素材 | プロンプト例 |
|------|-------------|
| メインビジュアル | ボタニカルイラスト風の背景に、窓辺で本を読む猫のシルエット。セージグリーンとベージュの配色。ノスタルジックで温かみのある雰囲気。水彩タッチ |
| 猫イラスト（ワンポイント） | シンプルな線画スタイルの猫。植物の葉と一緒に描かれている。ミニマルでおしゃれなデザイン |
| ボタニカルフレーム | 草花と蔦で構成された長方形のフレーム。ヴィンテージ植物図鑑風。背景透過用の線画スタイル |
| 背景テクスチャ | 古いクラフト紙のテクスチャ。わずかにクリーム色。経年変化した温かみのある質感 |
| セクション区切り | ボタニカルな葉と小さな花で構成された水平の装飾ライン。シンプルで上品なデザイン |

### 7.4 ボタニカルイラスト フリー素材サイト

| サイト名 | URL | 特徴 |
|----------|-----|------|
| unDraw | https://undraw.co/ | シンプルなSVGイラスト。色変更可能 |
| Biodiversity Heritage Library | https://www.biodiversitylibrary.org/ | パブリックドメインのヴィンテージ植物画 |
| rawpixel | https://www.rawpixel.com/ | パブリックドメインのボタニカルアート多数 |
| Pixabay | https://pixabay.com/ | 商用利用可の写真・イラスト |
| FreePik | https://www.freepik.com/ | ボタニカルイラスト豊富（要帰属表示の素材あり） |
| イラストAC | https://www.ac-illust.com/ | 日本語対応。商用利用可 |

### 7.5 素材制作フロー

1. **素材リストの確定** - 各ページで必要な素材を洗い出し
2. **AI生成 or フリー素材検索** - メインビジュアル・装飾はNano Bananaで生成、フレーム・パーツはフリー素材サイトで調達
3. **編集・調整** - 色調補正、サイズ調整、背景透過処理など
4. **最適化** - WebP形式への変換、圧縮、遅延読み込み設定
5. **サイトへ実装**
